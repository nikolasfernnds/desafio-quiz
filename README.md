# desafio-quiz

App de quiz em Kotlin com Jetpack Compose, com perguntas de múltipla escolha e feedback imediato.

## Funcionalidades
- Perguntas com alternativas  
- Feedback (certo/errado)  
- Navegação entre perguntas  

## Tecnologias
- Kotlin  
- Jetpack Compose
